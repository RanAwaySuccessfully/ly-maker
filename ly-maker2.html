This is a tool that generates layouts usable in SMWCentral (they can be used elsewhere, but they may not display correctly). Any changes to the layout are instantly shown on the layout preview window. Your layout must follow SMWC's layout rules if you actually want to use it. If you use any of the signatures generated here, you should disable the signature/footer separator in Edit Profile.

Make sure your layout's colors aren't eye-searing by checking them in this contrast ratio tool. When manually inserting color values, they must be in either of these formats: #00FF00 or #0F0 / rgb(0, 255, 0) / hsl(120, 100%, 50%). Transparency (alpha) values should be inserted into the neighboring opacity fields as percentages (from 0 to 100), when available.

If the post doesn't occupy the entire post frame (there's empty space at the bottom), change the min-height: ???px; property in your post layout to a higher value.
